# HardwareMon Website

The official website for **HardwareMon** a modern cross-platform system monitoring application built with Flutter and Python.

## Features

- Responsive modern design
- SEO optimized pages
- Screenshot gallery
- Cloudflare Pages hosting
- Fast static performance
- Clean lightweight layout

## Pages

- `index.html` — Main landing page
- `screenshots.html` — Application screenshots
- `sitemap.xml` — SEO sitemap
- `robots.txt` — Search engine rules

## Tech Stack

- HTML5
- CSS3
- Cloudflare Pages
- GitHub

## Deployment

The website is automatically deployed through Cloudflare Pages whenever updates are pushed to the repository.

## SEO

The website includes:

- Sitemap support
- Semantic HTML
- Open Graph metadata
- Mobile responsive layout
- Google Search Console integration

## What is HardwareMon?

HardwareMon is a cross-platform system monitoring application focused on modern UI design, live hardware statistics, and lightweight performance monitoring across Linux and Windows.
